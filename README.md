## 📌 Project Overview
This project focuses on predicting product defects in a manufacturing environment using **Logistic Regression**.  
The goal is to identify defective products early, reduce production costs, and improve overall quality through data-driven insights.

---

## 🎯 Business Objective
- Detect defective products in advance  
- Minimize production losses and operational costs  
- Improve product quality using predictive analytics  

---

## 📊 Dataset Description
The dataset is created by combining multiple production-related sources:

- 🏭 Production data (operational metrics)  
- ⚙️ Machine master data  
- 👨‍🔧 Operator master data  
- 🎯 Target variable: `defect_flag` (Defective / Non-defective)

---

## 🛠️ Data Preparation
- Merged multiple datasets into a unified dataset  
- Combined machine and operator details with production data  
- Performed data cleaning and preprocessing  
- Conducted Exploratory Data Analysis (EDA)  

---

## 🤖 Model Used
### Logistic Regression
- Selected for:
  - ✔️ High interpretability  
  - ✔️ Strong baseline performance  
  - ✔️ Simplicity and efficiency
